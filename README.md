Uncle Property - Static HTML Site (Black & Gold)
-----------------------------------------------
Files:
- index.html
- styles.css
- properties.json

Cara penggunaan:
1. Tempatkan ketiga file di folder yang sama.
2. Untuk hasil terbaik, host di server statis (Vercel, Netlify, GitHub Pages) atau jalankan simple HTTP server:
   - Python 3: `python -m http.server 8000`
   - Node (http-server): `npx http-server . -p 8000`
3. Buka http://localhost:8000 atau URL hosting Anda.
